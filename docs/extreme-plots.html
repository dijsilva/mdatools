Extreme plots

Extreme plot was proposed by A. Pomerantsev and O. Rodionova as a tool for comparing methods for computing of critical limits as well as for optimization of classification model parameters, e.g. number of components. The idea of the plot is following. For any given subset of \(i\) objects from calibration set of size \(n\), we can compute \(\alpha\) as \(\alpha = i / n\) and count how many objects have p-values smaller than the \(\alpha\).

In ideal case the number of such objects should be the same as \(i\), but in fact it can be in interval \(i± 2\sqrt{i (1 - \alpha)}\). For example, if \(i = 10\) and \(n = 50\) like in our last model, \(\alpha = 10/50 = 0.2\). So we can expect \(10±2\sqrt{10 * 0.8} = 10±8\) objects may have the p-value smaller than \(0.2\).
